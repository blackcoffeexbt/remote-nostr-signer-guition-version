{\bfseries{Connecting Applications\+:}}
\begin{DoxyEnumerate}
\item Applications use the bunker URL to initiate NIP-\/46 connections
\item Device prompts for user authorization of new connections
\item Approved applications are stored in authorized clients list
\end{DoxyEnumerate}

{\bfseries{Signing Events\+:}}
\begin{DoxyEnumerate}
\item Connected applications send signing requests via NIP-\/46
\item Device displays event details and prompts for user confirmation
\item User approves or rejects each signing request individually
\item Private key never leaves the device
